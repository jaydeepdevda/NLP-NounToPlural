# NLP-NounToPlural
Convert Noun to Plural Rule based simple Algorithm in python
noun to plural

10 Examples:

ordinary ""+'e'
1. tree -> trees
2. lake -> lakes
3. window -> windows

if 'y' then replace y with 'ies'
4. story -> stories
5. butterfly -> butterflies

if 'ss' then then ""+'es'

6. glass -> glasses
7. wish -> wishes
8. pitch -> pitches
9. bus -> buses
10. box -> boxes


When the base word ends with ch, sh, s, ss, x, z, add (es) to make it plural.
